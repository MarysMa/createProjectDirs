# createProjectDirs
quickly to create dirs for normal project



## Installation

```javascript
	$ npm i create-project-dirs -g
```
## How To Use

Enter your project dir and execute the cmd:

```javascript
	$ create -nm
```
## Docs

```javascript
	-nm  --------- create the project dirs of normal mobile
	-mp  --------- create the project dirs of normal pc
```
